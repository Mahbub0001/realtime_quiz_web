# Real-time Quiz Platform Frontend

## Setup

1. Navigate to `frontend/` directory.
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`

Make sure the backend is running at `http://localhost:8000` (the default API URL is configured in `src/api/client.js`).

## Usage

- Teacher login is at `/teacher/login`
- Students join at the root `/` with the Game PIN.
